# Code of Conduct

PAHLAWAN ROLEPLAY welcomes focused, respectful, and safety-minded collaboration.

## Expected Behavior

- Keep discussion respectful and relevant to the project.
- Prefer small, reviewable changes over broad rewrites.
- Follow the docs, OpenSpec scope, and validation checklist before requesting review.
- Report security concerns privately through `SECURITY.md` instead of posting secrets or exploit detail publicly.

## Unacceptable Behavior

- Harassment, insults, threats, or targeted personal attacks.
- Publishing credentials, tokens, cookies, OTPs, database dumps, private logs, or private user data.
- Encouraging destructive changes, unauthorized database writes, or unsafe runtime operations.
- Repeatedly pushing broad changes that ignore project scope and maintainer guidance.

## Reporting

For conduct issues, contact the maintainer through the currently agreed private project channel. For security issues, follow `SECURITY.md`.

## Enforcement

Maintainers may edit, hide, close, or remove issues, pull requests, comments, or contributions that violate this code of conduct or expose sensitive data.
